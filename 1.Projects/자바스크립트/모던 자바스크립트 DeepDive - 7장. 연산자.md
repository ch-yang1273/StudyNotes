# 7장. 연산자

## 7.3 비교 연산자

```javascript
// 동등 비교
if(x == y) // x와 y의 값이 같음

// 일치 비교
if(x === y) // x와 y의 값과 타입이 같음
```

- 동등 비교 연산자는 사용하지 않는 편이 좋다.

## 7.9 지수 연산자

- __지수 연산자는 ES7__ 에서 도입되었다. 쓰지 말고 `Math.pow(x, y)` 사용합시다.

```javascript
2 ** 2; // 4
Math.pow(2, 2); // 4
```
