키워드:: [[CSS101_8.Typography]]

line-height도 `px, em, rem` 단위를 사용한다.

line-height는 폰트에 비례한 줄 간격을 설정하기 위해 **`em` 단위을 많이 사용**한다.

`em`을 단위로 사용할 때는 단위를 적지 않아도 된다. (관례)

```css
.text {
    font-size: 16px;
    line-height: 1.5;
}
```

line-height를 크게 해도 **글자는 항상 세로 가운데** 오기 때문에 수직 정렬에도 많이 쓴다.
